# Роль для настройки journald на серверах pgsql

Настройки уменьшают место в оперативной памяти, занимаемое данными из журнала событий journald.  
То есть освобождают оперативную память под использование прикладным ПО.  
Задача в Jira: ["Настроить хранение данных журналов событий journald на диске для pgsql серверов (>1000 шт.)"](https://jira.tcsbank.ru/browse/IN-1160).

## Замечания
Роль находится в состоянии разработки.
